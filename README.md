# Welcome to your App  

This is a custom web application built with React, TypeScript, and Tailwind CSS. It features a dynamic character customization system, real-time animations, and a responsive UI.  

## Features  
- **Character Customization**: Choose from 24 unique apron designs for different actions (idle, walk, run, etc.).  
- **Interactive Controls**: Use keyboard shortcuts (WASD, SHIFT, SPACE, etc.) to control character movements and actions.  
- **Responsive Design**: Optimized for mobile and desktop with smooth transitions and pixel-perfect animations.  

## How to Use  
1. Clone the repository.  
2. Run `npm install` to install dependencies.  
3. Start the development server with `npm run dev`.  
4. Open the app in your browser and explore the customization menu.
